# electron-babel6
A electron app template to use babel v6

## using node version
- v5.6.0

## Start developing
install electron, gulp and modules

```
npm install -g electron-prebuilt gulp
npm install
```

build es6 file
```
gulp compile
```

## memo
- use shrinkwrap
```
npm shrinkwrap --dev
```
